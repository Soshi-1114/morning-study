# 朝の学習クイズ システム

毎朝、Claude Code Routine（クラウド実行）が自動で学習クイズを出題し、
前日の採点結果からあなたの弱点ジャンルを集計して知識マップを更新します。
PCがオフでも動き、PC・スマホ両方のGitHubアプリで問題を閲覧・採点できます。

## 仕組み（毎日のループ）

```
朝7時（ルーチン実行・クラウド）
  ├─ quizzes/ の過去シートのチェックを集計
  ├─ progress.md の知識マップを更新（弱点ジャンルを可視化）
  └─ 今日のクイズ quizzes/YYYY-MM-DD.md を生成（弱点を多めに出題）
        ↓
日中（あなた・PCでもスマホでも）
  ├─ GitHubで今日のシートを開く
  ├─ 自分で解く → 解答を開いて答え合わせ
  └─ 正解した問題の - [ ] を - [x] にする
        ↓
翌朝（ルーチンがこのチェックを読んで再集計）
```

## ファイル構成

| ファイル | 役割 |
|----------|------|
| `quizzes/YYYY-MM-DD.md` | 1日分のクイズシート（チェックボックス＋折りたたみ解答） |
| `progress.md` | 知識マップ・難易度レベル・集計ログ（ルーチンが自動更新） |
| `ROUTINE_PROMPT.md` | ルーチンに貼り付けるプロンプト |

## セットアップ手順

### 1. このフォルダをGitHubリポジトリにする
このリポジトリをGitHubにプッシュする（例：`morning-study`）。

### 2. Claude Code on the web を有効化
- 有料プラン（Pro / Max / Team / Enterprise）が必要。
- claude.ai/settings でWebアクセスを有効化し、GitHubを連携する。

### 3. ルーチンを作成
- claude.ai/code/routines を開く（またはCLIで `/schedule`）。
- **Prompt**：`ROUTINE_PROMPT.md` の中身を貼る。
- **Repository**：このリポジトリを選択。
- **Trigger**：スケジュール → 平日 朝7時（ローカル時刻でOK）。
- **Run now** でテスト実行し、今日のシートが生成されることを確認。

### 4. 採点の運用
- 問題を解いたら、正解した設問の `- [ ]` を `- [x]` に変える。
- スマホのGitHubアプリでもタップで切り替え可能。
- 翌朝のルーチンがこのチェックを読み、弱点を集計する。

## 注意点

- ルーチンはデフォルトで `claude/` プレフィックスのブランチにのみ push する。
  クイズを main に直接置きたい場合は、ルーチンの権限設定の調整、または
  生成PRを自動マージする運用が必要。（リサーチプレビューの仕様は変わるため
  最新は code.claude.com のドキュメントで確認すること）
- 1アカウントあたり1日の実行回数に上限がある（毎朝1回なら問題なし）。
- 機能名・提供範囲・ベータ仕様は更新されやすい。設定前に公式ヘルプで確認を。
